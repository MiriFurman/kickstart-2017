Basic HTML Document
===================

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>FED Bootcamp</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>
```

- `DOCTYPE` - Defines the dialect of the HTML we are writing
- `html` - The root of the document
- `head` - Defines settings for the browser
- `body` - The content of our document
